Il y a quatre raisons possibles pour lesquelles ce message n'existe pas :

  1. L'URL est incorrecte : Vérifiez à nouveau l'URL et assurez-vous de ne pas avoir fait d'erreur.
  2. Le message a expiré : L'expéditeur définit une durée de validité pour le message.
  3. Vous essayez de voir le message une deuxième fois : Désolé, nous supprimons le message immédiatement après le premier accès. Cela garantit uniquement la confidentialité du message.
  4. Quelqu'un d'autre a vu le message avant vous : Si vous essayez de voir le message pour la première fois, en utilisant la bonne URL et pendant la période de validité, cela signifie que quelqu'un d'autre a vu le message avant vous. Vous devriez immédiatement prendre des mesures pour atténuer les conséquences éventuelles de la lecture du message par quelqu'un d'autre. Nous vous suggérons de contacter l'expéditeur du message pour l'informer de la violation de la confidentialité.
